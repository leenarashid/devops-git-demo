#Instructions about SSH-connection 

1)After cloning the repo ->do cd ~ -> cd .ssh 

2)Inside .ssh,generate keys through ssh-key-gen

3)Inside .shh also vim config -> Add three things 9(1)Host github.com,(2)AddKeysToAgent yes, (3)IdentotyFile ~/.ssh/<your key_name>)

4)Add public key, generated through ssh-key-gen to SSH and GPG on your GitHub.

5)cd your repo folder
6)Do git remote -v
7)Do git remote set-url origin https://<your SSH URL>@github.com<your_name><your_repo>.git
8)Make maodifications, do git add
9)git commit -m "your_message"
10)git push origin main 

#IMPORTANT:In the process make sure that the name of the keys in .ssh-> config file is same as your generate keys
