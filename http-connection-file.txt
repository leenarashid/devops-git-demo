
#Instructions To Follow To Connect Through HTTP Connection

1)Make a folder
2)cd folder
3)git clone <repo-code>
4)ls repo-folder
5)Now check the url with git remote -v
6)Go to Settings -> Developer Settings -> Generate Classic Token(PAT)
7)git remote set-url origin https://<PAT>@github.com<your_name><repo_name>.git
8)git add <your_file>
9)git push origin main

#The above command will push all the modifictaions to the repo.

10) Changes can be verified through git status
